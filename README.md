Проверка аутентификации и авторизации пользователя


Предварительные требования
- Python 3.8+
- pip (менеджер пакетов Python)
- Google Chrome (браузер для тестов)

Клонирование репозитория
- git clone https://github.com/pikampg/etecs_test.git
- cd etecs_test

Установка зависимостей
- pip install -r requirements.txt

Запуск тестов
- pytest -v

Тесты
- test_open - проверка открытия сайта
- test_successful_login - тест успешной авторизации
- test_logout - тест выхода из системы
